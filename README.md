# 🐸 Stick (to the plan)

**Трекер привычек и осознанности для людей с СДВГ и нейроотличным восприятием.**

> «Не идеально, но достаточно»

Большинство трекеров задач спроектированы для нейротипичных людей: длинные «серии» (streak), строгий контроль и наказание за пропуски. Для людей с СДВГ это становится источником тревоги, а не помощи. **Stick (to the plan)** реализует концепцию *поддерживающей продуктивности* — помогает заботиться о себе без чувства вины и перегрузки. Центральный персонаж — лягушка-компаньон, которая сопровождает и подсказывает мягко, без осуждения.

🔗 **Живая версия:** https://stick-to-my-plan.vercel.app/

## ✨ Возможности

- **Облако тлеющих дел** — давно откладываемые задачи с лимитом и рулеткой случайного выбора «Мне повезёт!»
- **Микро-привычки** — маленькие шаги с возможностью «пропустить без вины»; подбор под уровень энергии дня
- **Банка желаний** — баллы за дела обмениваются на награды (в том числе guilty pleasures)
- **Уголок спокойствия** — дыхательные практики, заземление 5-4-3-2-1, гибкий Pomodoro
- **Сад идей** и **Социальный мост** — захват мыслей и мягкие напоминания о близких
- **Кнопка «Я сделал достаточно»** — разрешение остановиться
- **Геймификация без давления** — уровни, счастливый билетик, копилка добрых дел, сезонные подсказки
- Регистрация и авторизация, тёмная тема, экспорт данных

## 🛠 Технологии

**Frontend:** React 18, TypeScript, Vite, Zustand, Tailwind CSS, shadcn/ui, Framer Motion, React Router, Axios
**Backend:** Node.js, Express, Prisma ORM, PostgreSQL (SQLite в разработке), JWT, bcrypt, helmet, rate-limit
**Тесты и деплой:** Vitest, GitHub, Vercel (фронт), Render (бэк + база)

## 🚀 Запуск локально

Нужны установленные Node.js и база данных PostgreSQL (или SQLite для разработки).

**Бэкенд:**
```bash
cd stick-backend
npm install
# создайте файл .env:
#   DATABASE_URL="<строка подключения к базе>"
#   JWT_SECRET="<любая длинная случайная строка>"
npx prisma db push      # создаёт таблицы
npm run dev             # сервер на http://localhost:3000
```

**Фронтенд:**
```bash
cd froggy-focus
npm install
# создайте файл .env.development:
#   VITE_API_URL=http://localhost:3000/api
npm run dev             # приложение на http://localhost:8080
```

## 🗺 План развития

- **Аналитика** — статистика практик осознанности (минуты, прирост настроения) и социальные паттерны
- **Голосовой ввод идей** — надиктовать мысль через Web Speech API
- **PWA** — установка на телефон и работа офлайн
- **Push-уведомления** — мягкие напоминания от лягушки
- **Реальные бонус-баллы за месяц** — серверный маршрут начисления за достижение цели месяца
- **Полная синхронизация** привычек, идей и контактов с сервером
- Расширение покрытия тестами

## 📝 О проекте

Дипломный проект по веб-разработке. Концепция, дизайн и реализация — авторские;
лягушка-компаньон разработана самостоятельно. Разработка велась итеративно с использованием
ИИ-инструментов как помощников-менторов, при самостоятельной проверке и доработке каждого решения.
